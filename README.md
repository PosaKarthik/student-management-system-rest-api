# Student Management System REST API

A RESTful Student Management System built using Spring Boot, Spring Data JPA, MySQL, and Maven. This application provides APIs to manage student records with CRUD operations.

## Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Maven
- REST APIs
- Git & GitHub

## Project Status

🚧 Currently in Development

## Features

### Planned Features

- Add Student
- Get All Students
- Get Student By ID
- Update Student Details
- Delete Student
- Exception Handling
- ResponseEntity Support
- MySQL Database Integration

## Dependencies

- Spring Web
- Spring Data JPA
- MySQL Driver
- Spring Boot DevTools
- Lombok
